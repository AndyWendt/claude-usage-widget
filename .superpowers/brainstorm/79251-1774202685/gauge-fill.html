Implementation Approach

A

Custom NSImage (Recommended)

Programmatically draw the gauge into an NSImage at each refresh. Full control over arc, needle, and color. Renders crisp at any display scale. This is how apps like iStat Menus do it.

B

SF Symbol Palette Rendering

Use gauge.open.with.lines.needle.* SF Symbols with hierarchical or palette rendering to tint layers. Simpler code but limited — can't control fill level or needle position independently.
